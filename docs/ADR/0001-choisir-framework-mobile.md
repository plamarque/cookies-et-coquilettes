# ADR 0001 — Choix du framework mobile

## Statut

Proposé (non décidé)

## Contexte

Cookies & Coquillettes est une application mobile de cahier de recettes. Le choix du framework conditionne la stack, la structure du code et les outils de développement.

## Options envisagées

1. **React Native (Expo)** — Écosystème JavaScript/TypeScript, large communauté, développement rapide, hot reload.
2. **Flutter** — Dart, performances natives, UI riche, widgets personnalisables.
3. **Autre** — Capacitor, NativeScript, etc.

## Décision

[À décider] — À documenter une fois le choix effectué.

## Conséquences

- Définition de la structure des dossiers (src/ vs lib/).
- Choix des outils de persistance (AsyncStorage vs Hive, SQLite, etc.).
- Définition des commandes dans docs/DEVELOPMENT.md.
