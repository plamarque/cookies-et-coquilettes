# Screenshots Stores

Captures générées automatiquement par :

```bash
npm run screenshots
```

Arborescence cible :

```text
apps/web/public/screenshots/
  ios/
    iphone/
    ipad/
  android/
    smartphone/
    tablet/
```

Résolutions générées :

- iPhone : 1284x2778 (portrait)
- iPad : 2732x2048 (paysage)
- Android smartphone : 1080x1920 (portrait)
- Android tablet : 1920x1080 (paysage)
